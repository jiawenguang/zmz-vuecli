# 公告
``zmz-vuecli``  追梦猪vue脚手架全家桶减少重复性工作,提高工作效率,为快捷开发而生!（加q群:581764678）


## 特性

- 减少重复性工作
- 提高工作效率
- 快捷开发

## 安装

- npm

```bash
zmz-vuecli init <project name>

npm i或install
```
